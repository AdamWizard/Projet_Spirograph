\doxysubsection{Detailed Description}
Class for the discs that will build our \doxyref{Spirograph}{p.}{class_spirograph}. 

A \doxyref{Disc}{p.}{class_disc} object contains a sf\+::\+Circle\+Shape, an angular speed (to adjust its position every frame), and a list of \doxyref{Pencil}{p.}{class_pencil} objects to create multiple curves simultaneously. \doxyref{Disc}{p.}{class_disc} inherits from Virtual class \doxyref{Shape}{p.}{class_shape}
