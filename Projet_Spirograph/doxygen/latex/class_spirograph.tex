\doxysubsection{Detailed Description}
Main class that will keep track of every object position. 

A \doxyref{Spirograph}{p.}{class_spirograph} object is an \char`\"{}embedded\char`\"{} list of \doxyref{Disc}{p.}{class_disc} objects with a user-\/given size that will simulate the curves created by these Discs, even if the construction isn\textquotesingle{}t achievable in real life.
