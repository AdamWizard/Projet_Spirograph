Detailed Description

Main class that will keep track of every object position.

A Spirograph object is an "embedded" list of Disc objects with a user-given size that will simulate the curves created by these Discs, even if the construction isn't achievable in real life.

void Spirograph::update ( )

Main function to update the position of Discs within the spirograph.

This method will be called in each frame to adjust the coordinates of the Discs and pencils thanks to the mathematical equations describing the movement of the centers of all the circles relatively to each others
